# Carmelo's Agency - Sistema de Reserva de Viajes

Este proyecto es una plataforma integral de gestión de viajes diseñada para ofrecer una experiencia completa tanto a usuarios como a administradores, desarrollada bajo el framework Laravel.

## Características Principales

* **Buscador Avanzado:** Filtrado dinámico por título, destino, descripción y categorías relacionadas.
* **Sistema de Reservas:** Gestión de plazas y visualización de viajes contratados desde un panel de usuario.
* **Gestión de Favoritos:** Listado personalizado de destinos guardados por el usuario.
* **Sistema de Reseñas:** Los usuarios que han completado una reserva pueden publicar, editar y eliminar opiniones con valoración numérica.
* **Notificaciones de Usuario:** Implementación de SweetAlert2 para alertas de éxito, error y confirmaciones de acciones críticas.
* **Perfil de Usuario:** Panel de edición de perfil con soporte para carga y actualización de imágenes de avatar.

---

## Tecnologías Utilizadas

* **Backend:** PHP 8.x + Laravel Framework
* **Base de Datos:** MySQL / MariaDB
* **Frontend:** Blade Templates, CSS3 (Variables, Flexbox y Grid)
* **Interactividad:** JavaScript & SweetAlert2
* **Infraestructura:** Desplegado en AWS (Amazon Web Services) sobre entorno Ubuntu Server.
